⚙️ Setup 

Setting up the game is simple even if you’ve never 
worked with code before. Start by downloading the 
entire project folder from the repository. Every 
file inside that folder works together, so keep 
them all in the same place. Once the folder is 
on your computer, look for the file named 
Clicker_Game.html. This file is the main entry 
point for the game. Double‑click it, or right‑click 
and choose to open it in your web browser. The 
game will load instantly because it runs entirely 
inside the browser and doesn’t require installation, 
accounts, or extra programs. If the page doesn’t 
open, make sure you didn’t move or rename any of 
the other files, since the HTML page depends on 
them to load the game’s code and visuals correctly.

🕹️ How to Play 

When the game opens, you’ll see a main button or 
clickable area on the screen. Clicking this button 
earns you points, which act as the game’s currency. 
At first, each click gives you a small amount, but 
as you continue clicking, you’ll build up enough 
Clicks to buy your first upgrade. Upgrades appear 
on the screen with a cost next to them. When you 
purchase one, it increases the number of points 
you earn per click, letting you progress faster. 
This creates a simple loop: click to earn points, 
spend points to buy upgrades, and use those upgrades 
to earn points even more quickly. As your score grows, 
more upgrades become available, giving you new ways 
to increase your clicking power and experiment with 
different strategies. The goal is to keep improving 
your setup and see how quickly you can grow your total 
points over time.
